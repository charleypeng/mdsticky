# 更新日志

本项目的所有重要变更都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)。

## [1.1.0] — 2026-06-22

### 新增
- 多语言：运行时语言切换（10 种语言），通过 `tr()` + 按语言加载 `.lproj` 实现
- 显示模式：跟随系统 / 白天 / 夜晚
- 设置窗口：通用、同步、语言与外观标签页
- 同步服务：WebDAV、本地文件夹、SMB 后端，支持按服务配置
- 单实例强制启动，修复 macOS 14+ 弃用 API
- 关于面板添加作者信息（本地化标签）

### 变更
- 源代码语言从中文切换为英文，所有硬编码中文文本替换为英文
- 字符串目录 (.xcstrings) 重建，使用英文 key + 中文翻译
- 管理器窗口改为按需打开（启动时不显示），实现无打扰启动
- `environment(\.locale)` 替换为手动 `.lproj` 查表，保证 macOS i18n 可靠性

### 修复
- 应用图标增加内边距，适配系统显示
- 设置标签页改为工具栏风格 HStack 布局

## [1.0.0] — 2026-06-21

### 新增
- 工具栏：标题级别下拉（H1–H6）取代原先单一的标题按钮。选择级别后插入对应前缀，光标自动落在前缀末尾。
- 工具栏：代码块按钮插入围栏代码块，光标停在中间空行。
- 管理窗口：当前选中便利贴的 Markdown 实时预览（由 MarkdownUI 驱动）。
- 同步目标：WebDAV、本地目录、SMB 共享后端；基于文件系统事件，2 秒防抖。
- 菜单栏项目：「新建便利贴」「管理便利贴」「立即同步」「退出」。

### 变更
- Markdown 渲染引擎由原生 `AttributedString(markdown:)` 替换为 MarkdownUI，恢复完整 GFM 支持：标题、有序/无序/任务列表、表格、围栏代码块、引用、水平线、链接、图片、强调、删除线。
- 代码（行内与块）现在以半透明深色背景配白色文字渲染，在任何便利贴底色上都清晰可读。
- 工具栏回调重写为单一枚举动作类型（`.inline`、`.heading`、`.block`），调用方与测试共享一个类型。
- 管理窗口：详情面板为仅预览；编辑操作在悬浮便利贴窗口中进行。切换便签时通过 `.id(note.id)` 强制重建视图，保证预览刷新。

### 修复
- 工具栏按钮（加粗、斜体等）能正确包裹当前选中的文本（此前无效）。
- 标题下拉菜单项的颜色与其他工具栏按钮一致（之前使用系统强调色）。
- 便利贴窗口的编辑器在首次 view body 求值期间不再丢失 `NSTextView` 引用——此前会导致工具栏失效，需切换一次模式才恢复。
